# Shared Frontend Modules

Shared frontend utilities should live here:

- `i18n/`: Chinese/English dictionaries and language selection. Implemented.
- `format/`: bytes, rates, dates, durations, percentages, resource headroom text. Implemented.
- `status/`: status labels, pills, and dots. Implemented.
- `components/`: reusable panels, metric cards, compact rows, usage bars, and agent report rows. Implemented.

Keep feature-specific UI inside `features/`.
